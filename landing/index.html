Features

Every feature listed below is available for free in the offline local version.

Two ONNX Whisper Models

Fast mode uses whisper-tiny (~40MB) for quick results. Accuracy mode uses whisper-base (~74MB) for better context handling. Both run via WASM on your CPU.

Real-Time Streaming

See transcript chunks appear live as audio processes — no waiting for batch completion. 30-second chunked processing with 5-second stride overlap.

24+ Languages

English, Hindi, Arabic, Chinese, Spanish, French, German, Japanese, Korean, and 15+ more. Auto-detect mode identifies the spoken language.

Audio Merging

Combine multiple audio files into a single WAV track before transcribing. Uses Web Audio API — completely free, no limits.

Local History

All transcriptions saved locally in your app data. Search, re-open, and export past results. Copy all text or download as .txt.

Fully Offline

Internet needed once for license activation. After that, everything runs offline. Audio never leaves your computer. 7-day offline grace period.

OpenAI API Fallback

Optionally configure your own OpenAI API key for cloud-powered Whisper transcription. Automatic chunking for files over 25MB.

Dark & Light Themes

Toggle between dark and light mode. Theme preference persists across sessions. Designed for extended transcription work.

Technical specifications

System requirements, activation mechanics, and what runs under the hood.

Privacy Policy

CincoScribe Desktop does not collect analytics, telemetry, tracking pixels, or crash reports. The application operates 100% locally and does not transmit data back to any server.

Audio processing: All audio files are processed entirely on your local machine using bundled ONNX Whisper models via WebAssembly runtime. No audio is uploaded to any cloud server.

OpenAI Integration (Optional): If you optionally configure your own OpenAI API key in the settings, the app will make requests to OpenAI's API. This is completely opt-in.

Local storage: Your transcription history is stored locally on your device and is never shared or accessed remotely.
